## Usage

### Run Node.js server

```bash
cd server
```

```bash
pnpm i
```

```bash
pnpm dev
```

### Run React client

```bash
cd web
```

```bash
pnpm i
```

```bash
pnpm dev
```

Node.js server will run on port 3333, and React client on port 5173.
